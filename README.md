## 📖 项目简介

本项目是小说 **《药娘的天空》（原作仓库：[trans-archive/transky](https://github.com/trans-archive/transky)）** 的 **非官方**有声书版本。  
原作基于 [CC BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.zh-hans) 协议开放，本项目亦遵循相同协议。

> ⚠️**提示**
> 
> 本项目内容涉及 **跨性别群体的处境与社会话题**。  
> 请以尊重与理解的态度阅读与聆听。  
> 作品中的故事并非个人真实经历，请勿将其与现实人物混同。

## 🔊 听书方式

目前暂无在线播放版本。  
由于作品题材的特殊性，本项目暂不通过国内云盘等方式分享。

📦 如需收听，请：
1. 前往本仓库的 [Releases 页面](https://github.com/JuiceskyYeye/transky-audio/releases)；
2. 下载打包的 MP3 文件；
3. 使用任意音频播放器离线播放。

## 🎵 音频信息

- **语音合成平台**： [TTSMAKER](https://ttsmaker.cn/)  
- **使用声音**：默认声音 `1504-Xx潇潇`  
- **文件格式**：MP3  
- **后期处理**：无（直接生成音频，无额外剪辑或混音）  

> 若您希望修正某些章节的发音或内容错误，  
> 建议使用相同的声音 (`1504-Xx潇潇`) 重新合成，  
> 以保持整体声音风格一致。
