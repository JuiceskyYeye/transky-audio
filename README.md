## 📖 项目简介

本项目为开源小说 **《药娘的天空》（别名：*想要变成鹰的鱼*）**  
制作的 **非商业性有声书版本**。  
旨在让这部作品能以声音的形式被更多人听见与感受。

> 本项目遵循 [CC BY-NC-SA 4.0 协议](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.zh-hans)，

---

## 🌈 尊重与包容

本作品为**虚构文学创作**，并非作者个人真实经历。  
作品中涉及跨性别与少数群体主题，部分内容可能反映社会现实中的处境与心境。  
请以**尊重与理解的态度**对待作品中的人物与议题，  
避免使用歧视性、侮辱性或带有刻板印象的表达。
